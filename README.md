* 净化搜狗输入法.bat
  
* 净化微信输入法.bat

---------------------------------------------------

下面，给几点的建议：

* 首先，可以通过Glasswire或NetLimiter等防火墙软件，来验证，是否发生联网行为（再使用本脚本后，且重启后，本人电脑无连接情况）

* 其次，如果遇到无法删除的情况，请进入安全模式后，启用本脚本（或使用其他方法手工删除，尤其是搜狗的SogouExe.exe等程序）
